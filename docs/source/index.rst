socioFetcher
==============

A tool built for simplifying socio-economic data downloading process,
integrating python API wrappers of BLS_, BEA_, and Census_.
The tool also allows users to mapping the data in an interactive map
with ipyleaflet_ in Jupyter notebook.

.. _BLS: https://www.bls.gov/
.. _BEA: https://www.bea.gov/
.. _Census: https://www.census.gov/
.. _ipyleaflet: https://github.com/jupyter-widgets/ipyleaflet/
